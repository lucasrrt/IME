Quill Bill

API JSON para quill bill

RESTful API

Quill Bill utiliza o padrão REST para seu interfaceamento web.

Modelos

REST /users

Propriedades

Parâmetro Tipo Descrição
username string Nome de usuário, apenas letras miúsculas e números
password string Senha, quaisquer caracteres

Respostas

index Recupera uma lista com todos os usuários
show Recupera as propriedades de um usuário com id = user_id
update Altera as propriedades de um usuário com id = user_id
create Cria um usuário com as propriedades passadas via parâmetros query
destroy Deleta suário com id = user_id

REST /authentications

Parâmetro Tipo Descrição
username string Nome de usuário, apenas letras miúsculas e números
password string Senha, quaisquer caracteres

REST /bills

Parâmetro Tipo Descrição
user_id int Id do usuário
name string Nome da conta compartilhada, não precisa ser única. Pode receber qualquer string

REST /expenses
